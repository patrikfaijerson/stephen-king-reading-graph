How to use this

Each circle is a Stephen King novel or series entry. Lines show “read this first” or “this book can spoil that one”.

  • Click a book node to see details and suggested order.
  • Colors show how safe it is to start there without spoilers.
  • Use the toggles above the graph to hide links or show only spoiler-safe starting points.

This is a fan-made guide and not exhaustive. “Spoiler-safe” is a best-effort judgement, not a guarantee.
